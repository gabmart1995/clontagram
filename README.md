# Clontagram
Diseno de red social construido con nestjs framework.

### requisitos:
node >= 12.0
mysql o mariadb >= 10.0

## Instalacion

```bash
$ npm install
```

## Correr the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Pruebas unitarias

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
